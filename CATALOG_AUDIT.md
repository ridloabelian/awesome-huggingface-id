# Catalog Audit

Audit date: 2026-07-18.

## Scope

Audit ini mencakup repository GitHub yang dicantumkan dalam README. Pemeriksaan minimum: repository tersedia melalui GitHub API, nama canonical, archive status, stars, latest push, dan license metadata.

Status ini merupakan sinyal pemeliharaan, bukan security endorsement. Model dan dataset wajib diperiksa lagi pada model card/dataset card masing-masing sebelum dipakai.

## Governance

- Official: dimiliki organisasi Hugging Face atau organisasi upstream resmi.
- Community: dimiliki maintainer komunitas.
- Research: repository benchmark/paper dari kelompok riset.
- Archived/stale harus ditandai pada README saat terdeteksi.
- Link audit dijalankan ulang sebelum release besar.
