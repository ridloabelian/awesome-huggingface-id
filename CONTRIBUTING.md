# Contributing

Kontribusi dipersilakan melalui pull request.

## Syarat entri

- Relevan dengan Hugging Face, model, dataset, Spaces, inference, atau Bahasa Indonesia.
- Repository/resource masih dapat diakses.
- Deskripsi berdasarkan README atau dokumentasi resmi, bukan asumsi.
- Lisensi harus dicantumkan upstream atau status tanpa lisensi dijelaskan.
- Tidak berupa spam, affiliate link, mirror tanpa nilai tambah, atau proyek yang menyesatkan.
- Untuk dataset/model, jelaskan batas penggunaan komersial jika diketahui.

## Format

```markdown
- [Nama](https://github.com/owner/repo) - Deskripsi singkat yang diakhiri tanda titik.
```

Tambahkan entri ke kategori yang paling spesifik. Satu tautan cukup; jangan menduplikasi resource pada beberapa kategori.

## Verifikasi

Sebelum mengirim PR:

1. Buka setiap tautan.
2. Periksa README, tanggal update, dan lisensi.
3. Jalankan `npx -y awesome-lint`.
4. Pastikan deskripsi berbahasa Indonesia, objektif, serta berakhir dengan tanda baca.

Dengan berkontribusi, Anda menyetujui kontribusi daftar ini dirilis di bawah CC0.
